# Personality Prediction Analysis

## Project Overview

This project predicts personality types using Machine Learning techniques based on behavioral and demographic attributes.

The project covers the complete Data Science workflow including:

- Data Collection
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Building
- Model Evaluation
- Power BI Dashboard Development

---

## Objective

To analyze personality-related data and build a machine learning model capable of predicting personality categories accurately.
